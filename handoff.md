# Handoff — Silpo Nutri-Score demo

Контекст для нового чату. Повна архітектура репозиторію — у [`CLAUDE.md`](CLAUDE.md), тут лише стан і що робилось.

## Що це

Статичний HTML/CSS/JS прототип категорії "Здорове харчування" для Сільпо з фічею Nutri-Score (лейбл харчової якості A–E, зараз в даних тільки A/D) + бейджі поживних тегів (без цукру, алерген горіх, веган, без глютену, без лактози, органічний, багато білка).

Є **дві паралельні демо-версії** для порівняння дизайну на живому сайті:
- `index.html` → `product.html` — пігулка Nutri-Score
- `index-2.html` → `product-2.html` — квадратний Nutri-Score
